# DZMCycleScrollView

***
#### 简介:
    ScrollView 无限滚动 支持定时器轮播图 支持传入任何View以及Controller进行轮播 支持手动选定动画可选

    UIPageControl可在继承该控件之后自己添加即可 有代理回调 下面有例子
